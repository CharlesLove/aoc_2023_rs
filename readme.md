# Instructions
## To test:
`cargo test --bin day-xx`

## To run:
`cargo run --bin day-xx`
or
`cargo run --bin day-xx --release`
for better performance

## To benchmark:
`cargo bench -p day-xx`
(this appears to use the --release flag)
